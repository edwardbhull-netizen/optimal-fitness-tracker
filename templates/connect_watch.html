Apple Watch — iOS Shortcuts

1
Open Shortcuts app → tap + → name it Sync Steps to Optimal
2
Add Action → search Health → Get My Steps → set period to Today
3
Add Action → Get Contents of URL. Set URL to:
{{ base_url }}/api/steps
Method: POST. Body fields:
• steps = Shortcut Input
• token = your token
• source = Apple Watch
4
Save. Go to Automation tab → + → Time of Day → 10:30pm daily → Run Immediately → select your shortcut
5
Want hourly syncing? Repeat step 4 and add automations at 8am, 12pm, 4pm, 8pm — each runs the same shortcut. Steps update throughout the day automatically.
6
✅ Done. You can also tap Sync Steps Now on the Activity page for an instant manual sync anytime.

Apple Watch — Workout Sync

1
Open Shortcuts → + → name it Sync Workout to Optimal
2
Add Action → search Health → Find Workouts → set to Most Recent 1
3
Add Action → Get Details of Health Sample. Add three of these actions — one for Active Energy Burned, one for Average Heart Rate, one for Maximum Heart Rate
4
Add Action → Get Contents of URL. Set URL to:
{{ base_url }}/api/workout/sync
Method: POST. Body fields:
• token = your token
• calories = Active Energy value
• avg_hr = Avg Heart Rate value
• max_hr = Max Heart Rate value
• source = Apple Watch
5
Save. Go to Automation tab → + → Workout → Ends → any workout type → Run Immediately → select your shortcut
6
✅ Done. Finish a workout on your watch → calories and HR appear in the app automatically.

Garmin — Via Apple Health

1
In Garmin Connect → Settings → Apple Health → enable Workouts sync
2
Follow the Apple Watch workout setup above — once Garmin syncs workouts to Apple Health, the same shortcut picks them up
3
Note: there may be a short delay (1–2 mins) while Garmin syncs to Apple Health after a workout
